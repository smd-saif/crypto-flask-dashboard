# Crypto Overview Dashboard

## Overview

The Crypto Overview Dashboard is an interactive data analytics project that provides insights into cryptocurrency market performance using historical data. It enables users to analyze trends, compare key metrics, and visualize market performance through an intuitive dashboard.

## Features

* Interactive dashboard
* Cryptocurrency price analysis
* Market capitalization analysis
* Trading volume insights
* Historical trend visualization
* Interactive filters and slicers
* Clean and user-friendly dashboard design
